How to Install
=================

- To download the project follow the instructions bellow:

```bash
1. git clone 
2. cd ignite-challenge-blog
```

- Install the dependencies and start the server:

```bash
3. yarn
4. yarn dev
```

- or

```bash
3. npm install
4. npm dev
```
